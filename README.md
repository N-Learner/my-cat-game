## ✨ 核心特性

### 1. 动态弹弓与“多汁”物理手感 (Juiciness)
不再是僵硬的堆砌！我们实现了“所见即所得”的**动态锚点弹弓**，支持手指避让偏移。加入 **Squash & Stretch（挤压拉伸）** 动画，配合 128 BPM 节拍器，每一只猫咪落地都 Q 弹有力。

### 2. 双 BGM 场景切换：治愈 vs 鬼畜
- **封面态**：温馨治愈的《猫的舞步》，带你进入安静的猫咪世界。
- **游戏态**：瞬间切换至硬核重低音《Montagem Miau》，全场特效（屏幕脉冲、飘字、粒子）随 **128 BPM** 鼓点狂欢。

### 3. 吸血鬼幸存者式数值成瘾
- **指数级计分**：分数随层数平方级增长，配合 Combo 倍率，体验数值瞬间爆炸的爽感。
- **局外养成**：收集“猫罐头”，在商店购买**肉垫吸附（摩擦力）**、**猫神护符（复活）**等永久强化，每一局都能变强！

---

## 🛠️ 技术实现
- **引擎**：纯原生 HTML5 Canvas 实时渲染。
- **物理**：自研简化物理模拟（重力、碰撞、摩擦、弹性），支持局外参数注入。
- **音频**：Web Audio API 驱动，支持 iOS 引导式解锁与双音轨无缝 Crossfade。
- **存储**：使用 LocalStorage 实现全自动的局外养成数据持久化。

---

## 🕹️ 操作指南
1. **封面解锁**：点击跳动的声音图标，唤醒温馨旋律。
2. **蓄力发射**：在屏幕任意位置按下并拉动弹弓，观察预测轨迹。
3. **追求 Perfect**：让猫咪中心对齐，触发金色闪光与高额 Combo。
4. **商店强化**：失败后不要气馁，用赚来的罐头升级属性，挑战更高纪录。
